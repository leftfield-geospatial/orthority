Command line
============

Orthority command line functionality is accessed with the |oty|_ command, and its sub-commands.

Get help on ``oty`` with:

.. code:: shell

   oty --help

and help on an ``oty`` sub-command with:

.. code:: shell

   oty <sub-command> --help

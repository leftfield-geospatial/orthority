Image RPC tags and sidecar files
================================

:ref:`RPC camera <background/camera_models:rpc cameras>` parameters can be read from image tags or :file:`.RPB` / :file:`_RPC.TXT` sidecar files.  This capability is provided by the Rasterio_ / GDAL_ package.  See the `GDAL <https://gdal.org/development/rfc/rfc22_rpc.html>`__ and `GeoTIFF <http://geotiff.maptools.org/rpc_prop.html>`__ RPC documentation for details.
